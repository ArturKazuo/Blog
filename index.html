Início

Bem vindo(a) ao meu blog!
Este site foi criado para um trabalho de uma das matérias do curso de Ciência da Computação na Puc Minas. Meses depois, eu resolvi completamente reformulá-lo, adicionando, se possível, as novas tecnicas que aprendi desde então. Além disso, o objetivo principal deste blog é deixar à amostra todos os outros projetos que desenvolvi ou participei do desenvolvimento. Caso deseje entrar em contato comigo, basta clicar nos ícones do instagram e/ou email no canto superior direito da tela. Meu perfil no GitHub também está disponível para que todos possam acessar.

Projeto Tinkercad

Esta é a última versão de um projeto de LIP que os alunos fizeram na plataforma Tinkercad. Nele, programamos vários circuitos de arduíno para realizar alguma coisa. No meu caso, os LEDs vermelhos que podem ser vistos no canto superior direito acendem como se fossem um teclado, que toca a música "Ode To Joy" de Beethoven. O LCD ao lado dos LEDs mostra a música que está sendo tocada. Para acessar o projeto clique aqui
